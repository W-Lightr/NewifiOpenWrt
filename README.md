## 编译的Newifi d2 路由固件

适用于newifi d2 路由器的固件

基于 https://github.com/coolsnowwolf/lede 编译



#### 其他路由,更改配置config配置即可

###  配置：

1..在项目Settings -Secrets-New repository secret 添加名称 `RELEASE_TOKEN` 的token

`value`值点[这里](https://github.com/settings/tokens/new)获取

### 启动：

在Actions 启动workflows工作
